⚠️ Configuration requise

Pour que l'authentification fonctionne, vous devez configurer cette URL exacte comme Redirect URI dans votre application Twitch :

  1. Allez sur dev.twitch.tv/console/apps
  2. Créez une NOUVELLE application (ou utilisez une existante dédiée à l'overlay)
  3. Dans "OAuth Redirect URLs", copiez-collez exactement l'URL ci-dessus (sans espaces, sans slash final)
  4. Cliquez sur "Add" puis "Save"

⚠️ Si vous avez une erreur "redirect_mismatch" :

  • Vérifiez que l'URL dans Twitch correspond exactement à celle ci-dessus
  • Pas de slash final, pas d'espaces, pas de paramètres
  • Créez une application séparée pour l'overlay (pas celle du bot)
  • Attendez 1-2 minutes après avoir sauvegardé

⚠️ L'overlay est ouvert en local (file://). OAuth2 nécessite HTTPS. Déployez sur GitHub Pages pour utiliser cette fonctionnalité.
